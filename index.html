Arch Install and tech help

Installation guide consider unstalling vim and making a system link with sudo ln -s /usr/bin/nvim /usr/bin/vim vi was uninstalled and a link was made with nvim, vi=nvim now sudo ln -s /usr/bin/nvim /usr/bin/vi to remove system links use ls -l /usr/bin/pathToLink eg ls -l /usr/bin/vi and then do rm /usr/bin/vi

Also install os-prober package and run sudo os-prober with all things unmounted then run sudo grub-mkconfig -o /boot/grub/grub.cfg
