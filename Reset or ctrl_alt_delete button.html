Make a directory /etc/X11/xorg.conf.d/90-zap.conf containing

Section "InputClass" 

Identifier "keyboard defaults"
MatchIsKeyboard "on"

Option "XKbOptions" "terminate:ctrl_alt_bksp"

EndSection

the Option "XKbOptions" "terminate:ctrl_alt_bksp" is the same as running shell command

setxkbmap -option "terminate:ctrl_alt_bksp"
